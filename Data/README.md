# Dataset Description

This directory contains the AI evaluation dataset used in the
Engineering AI Evaluation Dataset project.

The dataset includes labeled AI-generated engineering responses,
correctness assessments, and categorized error types intended for
AI evaluation, data quality analysis, and reviewer training.
